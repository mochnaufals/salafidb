Mendengarkan Bacaan Ayat Al-Qur'an Secara Offline

Untuk melakukan hal ini anda harus mengunduh file audio yang ukurannya cukup besar ke harddisk anda. File-file audio tersebut dapat diunduh dari situs-situs berikut.
  1. Situs SalafiDB
  2. Situs Zekr
  3. Situs VerserByVerse

Cara Instalasi File Audio dari Situs SalafiDB

Situs Download SalafiDB menyediakan file audio Menshawi-11kbps dengan ukuran total file 180 MB. Untuk memudahkan download, file tersebut telah dipecah-pecah menjadi 9 file dengan masing-masing berukuran rata-rata 20 MB. Untuk unduh dan install lakukan hal-hal berikut.
  1. Buat direktori baru di harddisk anda.
  2. Pindah ke direktori Menshawi-11kbps dan unduh 9 audio file terpecah tersebut dan simpan di dalam direktori baru anda.
  3. Unduh file Join32.exe yang berada pada Situs Download SalafiDB dan simpan di dalam direktori yang sama.
  4. Klik dua kali file Join32.exe untuk menggabungkan file-file terpecah tersebut.
  5. Dari hasil penggabungan akan terbentuk file baru yang bernama: menshawi-11kb.exe.
  6. Klik dua kali file tersebut dan akan ditampilkan jendela untuk extract file.
  7. Tunjukkan letak direktori "salafidb/audio" anda. Gambar di bawah ini menunjukkan letak direktori tersebut pada drive C.

  
  1. Tekan tombol Extract dan file-file audio akan diletakkan dalam direktori "salafidb/audio/menshawi-11kb".
  2. Restart SalafiDB.
  3. Lihat menu: Preferences - Recitation dan pilih reciter Menshawi-11kb.


Cara Instalasi File Audio dari situs Zekr

Zekr adalah program Al-Qur'an yang dilengkapi dengan audio bacaan ayat Al-Qur'an juga. File audio yang digunakan oleh SalafiDB adalah kompatibel dengan file audio Zekr. Untuk unduh file audio Zekr lakukan hal-hal berikut.
  1. Buka IE/Browser anda dan kujungi situs Zekr Resource: http://siahe.com/zekr/resources.html
  2. Cari baigian "Offline recitations" pada halaman tersebut.
  3. Pada saat ini ada tiga file audio yang dapat diunduh yaitu:
  • minshawi.zip yang merupakan recitation oleh Menshawi dengan ukuran 180 MB. File ini adalah sama dengan file yang disediakan oleh situs SalafiDB bedanya file ini tidak dipecah-pecah.
  • ghamdi.zip yang merupakan recitation oleh Al-Ghamdi dengan ukuran 418 MB.
  • huzaifi.zip yang merupakan recitation oleh Al-Huzaifi dengan ukuran 290 MB.
  1. Download file audio di atas dan extract ke dalam direktori "salafidb/audio".
  2. Restart SalafiDB.
  3. Lihat menu: Preferences - Recitation dan pilih reciter yang telah anda unduh.


Cara Instalasi File Audio dari Situs VerseByVerseQuran
  
  1. Kunjungi situs http://www.versebyversequran.com dan lihat halaman download.
  2. Pada halaman tsb anda akan menjumpai folder-folder dengan nama para reciter misalnya Husary_128kbps.
  3. Setelah anda pindah ke folder reciter yang diinginkan maka akan dijumpai file-file MP3 dengan nama xxxyyy.mp3. Tiga digit pertama adalah nomor surat dan tiga digit terakhir adalah nomor ayat. Misalnya 001001.mp3 adalah file audio surat Al-Fatihah ayat 1.
  4. Anda dapat mengunduh sebagian atau seluruh file yang ada. Untuk mengunduh file-file audio yang jumlahnya ribuan anda dapat menggunakan salah satu aplikasi di bawah ini:
  • DownloadThemAll yang merupakan add-on browser Firefox (sangat mudah digunakan)
  • WebReaper
  • StarDownloader
  • GetRight
  1. Buat direktori baru dengan nama yang anda sukai (misalnya nama reciter "Husary-128kbps") di bawah direktori "salafidb/audio".
  2. Unduh file-file audio tsb dan simpan dengan tanpa merubah nama filenya ke dalam direktori baru tersebut.
  3. Restart SalafiDB dan temukan nama reciter yang merupakan nama direktori yang telah anda buat pada menu Preferences - Recitation.


Mendengarkan Bacaan Ayat Al-Qur'an Secara Offline

SalafiDB memungkinkan anda untuk mendengarkan bacaan ayat Al-Qur'an meskipun anda tidak mengunduh file-file audio di atas, dengan syarat anda harus terhubung ke Internet. Pada saat ini ada sepuluh reciter yang dapat anda nikmati secara online seperti yang ditampilkan pada gambar di bawah. (Catatan: bacaan ini dapat anda nikmati secara offline dengan mengunduh file audio dari situs VerseByVerseQuran seperti yang tertulis di atas.)
