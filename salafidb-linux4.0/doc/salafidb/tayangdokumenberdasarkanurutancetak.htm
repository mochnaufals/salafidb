Tayang Dokumen Berdasarkan Urutan di Buku/Sumber Asli
 Previous Top Next
Pada jendela hasil telusur, anda dapat menayangkan dokumen berdasarkan nomor urut pada sumber aslinya relatif terhadap dokumen yang sedang dilihat.

Cara:
  1. Lakukan pencarian dan pilih satu dokumen dari jendela hasil pencarian. Gambar di bawah ini menampilkan ayat ke-23 dari surat Al-Hajj beserta jendela pop-up-nya.

  
  1. Klik tombol mouse kanan pada jendela tampilan surat dan menu pop-up ditampilkan (lihat gambar di atas).
  2. Pilih perintah Previous untuk melihat ayat sebelumnya (yaitu ayat ke-22) atau Next untuk melihat ayat sesudahnya (yaitu ayat ke 24).
  3. Operasi ini dapat anda lakukan berulang-ulang. Misalnya sesudah memilih perintah Previous pada tampilan diatas, maka akan ditampilkan ayat ke-22. Pada jendela tampilan lakukan lagi perintah Previous maka akan ditampilkan ayat ke 21.
  4. Anda dapat juga melakukan operasi di atas dengan tombol shortcut. Aktifkan tampilan dengan mengklik jendela tampilan dan tekan tombol keyboard p untuk menjalankan perintah Previous atau tombol n untuk perintah Next.
