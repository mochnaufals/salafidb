Dengar Bacaan Ayat Al-Qur'an
 Previous Top Next
Fungsi: Mendengar bacaan ayat Al-Qur'an yang sedang dilihat baik pada jendela hasil telusur maupun pada jendela jelajah (Fungsi baru pada SalafiDB 4.0).

Cara Pemakaian:
Tampilkan satu ayat Al-Qur'an sebagai target dari jendela hasil telusur atau jendela jelajah Al-Qur'an. Kemudian tekan salah satu tombol di bawah ini yang terdapat pada toolbar.

: Tombol untuk mendengarkan bacaan ayat sekali saja.
: Tombol untuk mendengarkan bacaan satu surat dimulai dari ayat yang dipilih. Apabila fungsi ini dijalankan pada jendela hasil telusur, maka bacaan ayat akan dilakukan sampai surat terakhir Al-Qur'an.
: Tombol untuk mendengarkan bacaan ayat berulang kali.

Tombol-tombol berikut digunakan untuk mengontrol bacaan yang sedang didengar:

: jeda (pause) bacaan. Untuk memulai kembali tekan salah satu tombol di atas sesuai dengan tombol pada saat anda memulai bacaan ayat.
: menghentikan bacaan.

Catatan:
  • Anda dapat mendengarkan bacaan ayat Al-Qur'an secara offline dan online. Untuk mendengarkan secara offline anda harus terlebih dahulu mengunduh file audio dan menyimpan file tersebut di harddisk anda, sedangkan untuk mendengarkan secara online anda tidak perlu mengunduh file tsb tetapi cukup terhubung ke Internet saja. Untuk keterangan lengkap silahkan lihat File Audio Bacaan Ayat Al-Qur'an.
  • Fungsi ini juga bisa dijalankan melalui pop-up menu.
