Fungsi: Mencetak satu atau lebih dokumen.

Cara:
  1. Buka jendela editor cetak dengan memilih menu: Tools - Print...
  2. Seret-dan-jatuhkan (drag-and-drop) dokumen atau dokumen kategori dari jendela tayang ke jendela editor cetak. Misalnya gambar di bawah ini menyeret-dan-menjatuhkan ayat ke-5 dari surat Al-A'raaf ke jendela editor cetak.

 
  
  1. Hal-hal berikut dapat dilakukan untuk mengatur hasil cetakan:
  • Bila anda mencetak ayat-ayat Al-Qur'an, maka anda dapat meletakkan judul (nama surat dan nomor ayat) pada bagian bawah dengan memilih tombol At the bottom yang berada pada panel kontrol di jendela kanan.
  • Anda dapat menambah sejumlah jalur putus (break lines) diantara dua dokumen dengan menuliskan jumlah yang diinginkan pada kotak isian Breaklines.
  • Anda dapat menambah garis horizontal diantara dua dokumen dengan memilih HR Line check box.
  • Anda dapat membatasi satu halaman hasil cetak hanya memuat satu dokumen saja dengan memilih Newpage check box. Lihat catatan di bawah untuk menggunakan fungsi ini.
  • Anda dapat menghilangkan background tampilan dokumen dengan menghilangkan pilihan (unselect) Background check box.
  1. Dokumen yang telah diletakkan pada editor cetak dapat dihapus dengan terlebih dahulu memilih/mengklik dokumen tersebut dan kemudian menekan tombol Delete.
  2. Anda dapat menata-ulang (reset) editor cetak dengan menekan tombol Clear.
  3. Teks yang tertulis dalam kurung siku di editor cetak dapat dirubah dengan meng-klik dua kali (double click) teks tersebut. Anda dapat memasukkan kode HTML, seperti <html><h1>Surat Al-Fatihah</h1><html> pada teks tersebut.
  4. Untuk memindahkan letak suatu dokumen pada editor cetak, seret-dan-jatuhkan dokumen tersebut pada tempat yang diinginkan.
  5. Anda dapat menghilangkan atau mencantumkan bagian teks (misalnya teks transliterasi dan terjemahan bahasa Inggris pada ayat Al-Qur'an). Detail pelaksanaannya bisa dilihat di sini.
  6. Tekan tombol Preview & Print untuk pratilik hasil cetak dan mencetak pada printer atau tekan tombol Save as HTML untuk menyimpan kedalam format HTML. Keterangan lanjut tentang operasi Save as HTML lihat Fungsi Membuat HTML Dokumen.

Diizinkan untuk mempublikasikan isi SalafiDB dengan syarat tidak merubah isi/makna, mencatumkan sumber asli dan link SalafiDB.

Catatan:
  • Anda tidak dapat mencetak tafsir dengan editor cetak. Untuk mencetak tafsir, tampilkan tafsir (misalnya dengan membuka jendela jelajah tafsir), klik tombol . Anda dapat juga melakukan dengan klik mouse sebelah kanan pada jendela tafsir tersebut dan pilih menu print pada jendela pop-up.
  • Anda dapat menyeret-dan-menjatuhkan dokumen kategori, misalnya surat Al-Baqaroh, dan bahkan satu jenis dokumen, misalnya Al-Qur'an. Tapi perlu diingat, bahwa proses cetak ini dapat memakan waktu beberapa menit tergantung dari jumlah memori dan kecepatan CPU komputer anda.
  • Untuk mencetak kedalam format PDF, instal PDF printer driver dan pilih PDF printer tersebut pada saat mencetak. Anda bisa men-download PDF printer driver gratis dari sini PrimoPDF atau bisa ditemukan di sini.
  • Untuk cetak dengan opsi Newpage lakukan sebagai berikut.
  1. Pada jendela pratilik (preview), klik tombol mouse kanan untuk menampilkan jendela pop-up dan pilih perintah View with external browser.
  2. Isi pada jendela preview akan ditayangkan pada browser (misalnya Internet Explorer).
  3. Cetak tayangan tersebut dengan menggunakan fungsi cetak pada browser anda Misalnya pada Internet Explorer, pilih menu File - Print.
