Preferensi Analisa Kata Arab Al-Qur'an
 Previous Top Next
Anda dapat menggunakan metode Buckwalter untuk menayangkan hasil analisa kata Arab Al-Qur'an. Caranya adalah pilih menu Preferences - Change View - Buckwalter.



Catatan:
Fungsi ini menayangkan ejaan bahasa Arab dengan huruf latin. (Tayangan default adalah dengan menggunakan huruf Arab).
