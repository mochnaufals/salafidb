Fungsi Membuat HTML Dokumen
 Previous Top Next
Fungsi Cetak Dokumen memiliki fungsi untuk menyimpan dokumen-dokumen yang dipilih ke dalam format HTML. Ada dua fungsi yang berbeda:
  1. Menyimpan seluruh dokumen ke dalam satu HTML dokumen
  2. Menyimpan dokumen-dokumen masing-masing pada satu HTML dokumen. Fungsi ini juga membuat halaman indeks (index.htm) untuk mengakses HTML-HTML dokumen tersebut sehingga memudahkan anda untuk meletakkan file-file tersebut pada Web server dan membuat file CHM.

Berikut adalah contoh untuk melakukan fungsi nomor dua diatas:
  1. Jalankan Fungsi Cetak Dokumen dengan memilih menu: Tools - Print...
  2. Buka dokumen-dokumen yang akan ditulis ke format HTML misalnya buka jendela jelajah hadits Shahih Muslim.
  3. Seret akar node (root node) yang berlabel "Shahih Muslim" dan jatuhkan pada jendela Print Editor (lihat gambar di bawah ini)
  4. Seluruh hadits dan judul kitab akan ditampilkan pada jendela Print Editor (lihat gambar di bawah).

  
  1. Tekan tombol Save as HTML pada jendela Print Editor. Akan ditampilkan jendela dialog seperti gambar di bawah ini, dan tekan tombol Multiple files with index.

  
  1. Selanjutnya akan ditampilkan jendela dialog untuk menyimpan file. Masukkan nama direktori di mana file-file HTML yang akan dibuat disimpan. Gambar di bawah ini menampilkan jendela dialog penyimpanan file dimana nama direktorinya adalah Shahih-Muslim. Tekan tombol Save (maaf tombol pada gambar tertulis dalam bahasa Jepang.).

  
  1. Silahkan melihat file-file HTML tersebut pada direktori yang telah anda tentukan dan baca file index.htm (lihat gambar di bawah ini) dengan browser anda dengan mengeklik dua kali file tersebut.
  2. Anda dapat mempublikasikan kumpulan HTML file tersebut pada Web server, yaitu dengan mengopi seluruh file. Pada contoh diatas, kopi direktori Shahih-Muslim ke Web server anda.

Anda dapat dengan bebas mempublikasikan isi SalafiDB dengan syarat tidak merubah isi/makna, mencantumkan sumber asli dan link SalafiDB.
