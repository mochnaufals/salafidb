Q: Kenapa pada saat pertama kali SalafiDB dijalankan program terasa sedikit lambat

A: SalafiDB menggunakan kamus bahasa Arab untuk melakukan pencarian dan analisa kata bahasa Arab. Kamus tersebut ditaruh di memori pada saat pertama kali program dijalankan. Lamanya peletakkan kamus ke memori tergantung dari kecepatan CPU dan memori Anda. Jadi ada kalanya pada saat Anda melakukan pencarian awal, program masih sedang meletakkan kamus ke dalam memori sehingga pencarian tersebut terasa lambat.

Q: Bagaimana cara mencetak tafsir?

A: Klik tombol mouse kanan dan jendela pop-up akan ditampilkan. Pilih perintah Print....

Catatan:
  • Pada saat ini anda tidak dapat mencetak tafsir dengan editor cetak.
  • Cara diatas juga berlaku untuk jenis dokumen lain.

Q: Saya tidak dapat kopi-dan-pasta ke Word Processor (misalnya OpenOffice)

A: Buka dokumen tersebut dengan external browser dan lakukan kopi-dan-pasta dari external browser tersebut.
