Untuk mengupdate indeks lakukan hal berikut:
  1. Pilih menu Help - Update - Add Document Index.

  
  1. Akan ditayangkan jendela dialog. Pilih file indeks yang akan ditambahkan dan tekan tombol open.

  
  1. Setelah selesai akan ditayangkan pesan sukses.

Catatan:
  • Untuk melihat dokumen-dokumen yang ditambah ke index masukkan kata kunci berikut ke Window utama, indexdate:15082006 . 15082006 adalah time stamp index yang dapat dilihat dari nama index file yang didownload.
  • Silahkan memonitor situs SalafiDB  secara berkala untuk mengetahui penambahan data/indeks. Informasi tentang indeks/data baru bisa juga didapat dengan subscribe ke milis Salafi-DB.
